## 方法一

查看设备铭牌上的**产品型号/Model**，该项通常以产品系列名称开头。

## 方法二

SSH登陆设备后，使用命令**bm-smi**查询**Name**字段，这个字段是芯片名称和工作模式，以`BM1688-SOC`为例，芯片名称为`BM1688`，工作模式为`SOC`模式。

如下为常见芯片名称和产品系列的对应表

|芯片名称|产品系列|
|-|-|
|BM1684|SE5|
|BM1684|SE7|
|BM1688|SE9 16|
|CV186AH|SE9 8|
